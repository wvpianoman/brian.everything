# brian-everything

 This is my personal installation/maintenance scripts for **ArcoLinux** (*Arch*), Fedora **Rawhide**,  **Solus** and  openSUSE **Tumbleweed**.   I also have a ***NixOS*** repo here, but I do not use Nix any longer.
 (you will see a **MXLinux** (*Debian*) here, but I only have it as a backup/failsafe/woops
 
 If you can't tell, I prefer *Rolling Release* over Fixed (*point*) Release Distros.  This point isn't open to debate.  This repo fits my personal use-case, and might not be compatible with yours.

 I have taken inspiration from my brother from another mother, Tolga Erok

 You can find his GitHub here   https://github.com/tolgaerok

 He has collections available for NixOS, Debian and Fedora.
